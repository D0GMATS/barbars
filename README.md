# Северные Варвары — browser turn-based MMO RPG

Готовый MVP frontend + backend браузерной пошаговой онлайн RPG. Проект оригинальный: не использует название, графику, тексты, базу данных или код barbars.ru.

## Запуск

```bash
npm install
npm start
```

Откройте: http://localhost:3000

Для production задайте секрет:

```bash
JWT_SECRET="long-random-secret" PORT=3000 npm start
```

## Что есть сейчас

- Node.js/Express API + Socket.IO realtime.
- Регистрация, логин, JWT.
- Персонаж: уровень, опыт, золото, репутация, HP, энергия.
- Характеристики: сила, ловкость, выносливость.
- Производные параметры: урон, защита, уклонение, крит.
- Экипировка с редкостью, магазин, инвентарь.
- Расходники: зелья HP и энергии.
- Продажа предметов и материалов.
- Локации: поселение, лес, шахты, арена.
- PvE с масштабированием монстров под уровень героя.
- PvP-очередь и дуэли игроков.
- Пошаговый бой: удар, мощный, точный, ярость, защита.
- Добыча ресурсов в локациях с энергией и cooldown.
- Квесты и ежедневное задание.
- Лут с монстров, включая редкие предметы.
- Общий чат и лидерборд.
- Улучшенный адаптивный UI без внешних CDN.

## Структура

- `server.js` — Express API + Socket.IO + игровая логика.
- `public/index.html` — интерфейс.
- `public/style.css` — адаптивные стили.
- `public/app.js` — клиентская логика.
- `data/db.json` — JSON-хранилище, создаётся/мигрирует автоматически.

## Важно для production

Текущая JSON-база подходит для демо и прототипа. Для публичной MMO лучше перейти на PostgreSQL/Redis, потому что файловое хранилище контейнера может сбрасываться при redeploy, а JSON-файл плохо масштабируется на параллельную запись.
